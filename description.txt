Общее описание сервиса для главной страницы и для презентации:
HISS - сервис, который упрощает взаимоотношения пациента, мед. учреждения и страховой. В распределенной базе данных хранятся медицинские данные пациента. Шифрование и смарт контракты позволяют гарантировать защищенность, подлинность, анонимность записей. Историю болезни могут дополнять только те больницы, которым доверяет пациент. Страховая управляет страховкой пациентов при помощи смарт контрактов, может получить достоверную историю болезни и обращений к врачу.

Страница с контрактом пациента:
Ваши данные хранятся в зашифрованном виде, расшифровать их можно только при помощи закрытого ключа. Закрытый ключ хранится только у вас. При посещении больницы, вам необходимо предоставить расшифрованные записи.
Помните, что корректность расшифрованных вами данных легко проверяется.
Только клиники, в которых вы обслуживаетесь, могут оставлять записи в базе данных.
